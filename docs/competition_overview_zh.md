# AI+科研创新赛道项目说明（Bio-Diffusion-Science）

## 1. 项目简介

本项目基于几何完备扩散模型（Geometry-Complete Diffusion Model, GCDM），提供了一个用于 **3D 分子生成与性质优化** 的开源科研模型与工具链。
原始代码来自 Nature Communications Chemistry 2024 论文的官方实现，在此基础上，本仓库：

- 补充了面向中文科研用户的说明文档；
- 提供了简化的 Demo 脚本，方便快速生成示例分子；
- 明确给出了从环境创建、数据与模型下载、到实验复现的完整步骤。

## 2. 参赛方向对应关系

- **科研模型**：
  - GCDM 作为深度生成模型，可在连续时间扩散框架下生成 3D 分子结构，并支持性质优化。
- **科研工具**：
  - 提供命令行脚本与配置，研究者可直接调用预训练模型完成分子生成。
- **科研框架 / 工作流**：
  - 通过 `environment.yaml`、数据下载命令、Demo 脚本与评估流程，形成可复用的科研工作流雏形。

## 3. 技术实现概述

- **模型类型**：基于图神经网络与几何深度学习的扩散生成模型；
- **输入**：分子图及其 3D 坐标、属性标签（可选）；
- **输出**：符合物理化学约束的 3D 分子结构坐标与类型；
- **训练与评估数据集**：
  - QM9：小分子数据集，用于小分子生成与性质预测；
  - GEOM-Drugs：药物级分子数据集，用于大分子生成。

模型结构与训练细节完全遵循原论文与官方实现，详见 `README.md` 中 “Instructions for use” 与 “How to reproduce paper results” 部分。

## 4. 可复现实验与评估基准

本项目延续原仓库提供的复现实验说明，并在此基础上给出一个简化 Demo 作为示例：

- **简化 Demo（分子生成）**
  - 脚本：`scripts/science_demo_qm9.sh`
  - 作用：使用预训练的 QM9 模型无条件生成一批小分子样本。
  - 输出：`./science_demo_outputs` 目录下的 `.sdf` 文件，可用于后续可视化或性质分析。

- **论文级复现（可选，资源充足时执行）**
  - 无条件分子生成（QM9）
  - 条件分子生成（按物性条件）
  - 药物分子生成（GEOM-Drugs）
  - 分子性质优化实验  
  对应命令均已在 `README.md` 中列出，可根据计算资源选择性执行。

## 5. 实际科研场景中的应用验证（概念性案例）

1. **新材料 / 新分子设计**  
   研究者可设定目标物性范围（例如带隙、极化率等），使用条件生成或优化功能产生候选分子，后续再结合 DFT 计算或实验验证筛选潜在材料。

2. **药物发现早期筛选**  
   利用 GEOM-Drugs 相关配置，在药物级分子空间中快速生成结构合理的候选分子，再对接虚拟筛选、对接打分等下游工具。

3. **教学与方法学研究**  
   作为扩散模型与几何深度学习结合的典型案例，用于课程教学、方法对比实验、以及与其他生成模型（如 VAE、GAN）的比较研究。

在上述场景中，本项目提供了统一的代码与配置入口，可较为方便地与下游分析方法连接，实现 **“生成 → 分析 → 筛选”** 的基础科研工作流。

## 6. 使用方式概述

1. 按 `README.md` 或 `README_zh.md` 创建 conda 环境并安装依赖；
2. 根据需求下载对应数据集与预训练模型；
3. 运行 Demo 脚本或 README 中给出的命令；
4. 将生成的 `.sdf` 分子导入可视化或下游分析工具（如 RDKit、PyMOL 等）。
